# Development Standards

*This document describes conventions, approaches, practices, technologies and standards that are accepted and practiced by MLSDev. In other words, it describes how we do the things.*

*This document is updated frequently. Our practices and standards are subject to change as software development best-practices evolve.*

*Everyone who commits the updates this document should notify their colleagues of the changes made.*
